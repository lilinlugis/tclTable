# 两列表

两列表是指左侧有列冻结，右侧可滚动的表，两列表同样支持多行表头，与复杂表唯一不同的是不支持表头固定，表体滚动，这种表比较适合表体自适应高度，左侧有些列冻结，而表的列比较多的情况，如果没有列头结，建议使用多行表头的指令即可。

两列表的实现也是其于二维的列定义实现的。

```html
<div class="table-besides-columns" tcl-bisides-table="dataTable"></div>
```

上例中使用了tclTableService.initBisidesTable方法，tclTableService是服务于表格指令的AngularJS服务，它提供了一些列处理功能，这样可以简化配置，使用户专注于业务实现。

tclTableService.initBisidesTable方法会分析列定义中的pinLeft、宽度定义、以及列之间的colspan和rowspan。此方法拥有四种能力：
* 1、根据合并列的PinLeft来指定被合并列的pinLeft属性，
* 2、根据被合并列的列宽（默认100）来指定合并列的列宽，
* 3、初始化$$leftRows和$$rightRows属性来判断冻结和非冻结表要渲染的行数
* 4、初始化$$pinLeftWidth和$$contentWidth来为左右表格指定宽度$$wholeWidth为整表宽度，指令要监测整表宽度变化

目的：简化编程里的配置项，让程序来做自动补全的工作.这个方法有两个参数，第一个参数为原有的表格定义，第二个参数为表格的初始宽度，如果不传默认为window的宽度。在使用这种表格时，强烈建议使此方法做初始化的工作。
